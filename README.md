## 基本介绍

#### 开发技术 

- Vue3
- Ts
- Worker
- OffscreenCanvas
- Canvas 

#### 游戏描述

这是一款支持 `pc端` 和 `移动端` 的网页塔防小游戏。

## 项目启动

安装依赖
```bash
npm i
```

启动项目
```bash
npm run dev
```
